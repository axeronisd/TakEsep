Политика конфиденциальности

Дата последнего обновления: 4 мая 2026 г.

Настоящая Политика конфиденциальности (далее «Политика») описывает, какие данные собирают приложения TakEsep, AkJol и AkJol Pro (далее «Приложения»), как они используются и защищаются.

TakEsep Управление складом и продажами
Windows · Android · iOS
AkJol Заказ и доставка для покупателей
Android · iOS
AkJol Pro Управление доставками для курьеров
Android · iOS

1. Какие данные мы собираем

1.1. Данные учётной записи

При регистрации мы запрашиваем:

  • Номер телефона — используется как логин;
  • Имя пользователя (псевдоним) — отображается в интерфейсе;
  • PIN-код — для локальной аутентификации в приложении.

1.2. Данные о заказах и доставках

  • Содержание заказов (товары, количество, цены);
  • Адреса доставки и пункты назначения;
  • Статусы заказов и история изменений;
  • Информация об оплате (способ оплаты, сумма, скидка).

1.3. Геолокация

  • AkJol Pro — местоположение курьера в реальном времени для отслеживания доставки клиентом;
  • AkJol — местоположение клиента для выбора ближайшего магазина и расчёта стоимости доставки;
  • TakEsep — местоположение склада (при авторазведении).

Геолокация запрашивается только с вашего согласия. Вы можете отклонить запрос и использовать приложение без функций, зависящих от местоположения.

1.4. Данные устройства и уведомлений

  • FCM-токен (Firebase Cloud Messaging) — для отправки push-уведомлений;
  • Информация о сбоях и ошибках (через Firebase Crashlytics) — для улучшения стабильности приложений;
  • Модель устройства и версия ОС — для диагностики технических проблем.

2. Как мы используем данные

  • Для предоставления основных функций Приложений (оформление заказов, управление складом, доставка);
  • Для отправки уведомлений о статусе заказов и новых задачах;
  • Для отображения местоположения курьера на карте в реальном времени;
  • Для аутентификации пользователей и разграничения прав доступа;
  • Для аналитики сбоев и улучшения качества Приложений.

3. Передача данных третьим лицам

Мы не продаём и не передаём ваши персональные данные третьим лицам для маркетинговых целей. Данные обрабатываются следующими сервисами:

Сервис Назначение Данные
Supabase База данных, аутентификация, синхронизация Все данные учётной записи, заказы, товары
Firebase (Google) Push-уведомления, диагностика сбоев FCM-токен, данные о сбоях
PowerSync Офлайн-синхронизация (только TakEsep) Данные склада и товаров
OpenStreetMap Карты и геокодирование Координаты для отображения карты

Все перечисленные сервисы обрабатывают данные в соответствии со своими политиками конфиденциальности и применимым законодательством (GDPR, CCPA и др.).

4. Хранение и защита данных

  • Данные хранятся на серверах Supabase (регион — ЕС) в зашифрованном виде;
  • Передача данных между Приложением и сервером осуществляется по протоколу HTTPS/TLS;
  • Доступ к данным ограничен ролями (Row Level Security) — каждый пользователь видит только свои данные;
  • Локальные данные на устройстве хранятся в зашифрованном хранилище ОС;
  • FCM-токен удаляется при выходе из учётной записи.

5. Ваши права

Вы имеете право:

  • Доступ — запросить копию своих персональных данных;
  • Удаление — запросить удаление учётной записи и связанных данных;
  • Исправление — обновить неточные данные;
  • Отзыв согласия — отключить геолокацию и уведомления в настройках устройства;
  • Переносимость — получить данные в структурированном виде.

Для реализации любого из этих прав обратитесь к нам по контактам, указанным ниже.

6. Дети

Приложения не предназначены для лиц младше 16 лет. Мы сознательно не собираем персональные данные детей. Если вы узнали, что несовершеннолетний предоставил нам свои данные, свяжитесь с нами — мы удалим их.

7. Изменения Политики

Мы можем обновлять настоящую Политику. Актуальная версия всегда доступна по адресу:
https://axeronisd.github.io/TakEsep/privacy.html

Существенные изменения будут доведены до вашего внимания через уведомление в Приложении.
